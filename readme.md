# 🎭 Eventos da Cidade

Sistema web para divulgação e gerenciamento de eventos culturais desenvolvido em Python e Flask.

Projeto desenvolvido para a disciplina de Programação Web.

---

## 📖 Sobre o projeto

O sistema permite cadastrar e divulgar eventos culturais, oferecendo uma interface simples e intuitiva para consulta das informações dos eventos.

---

## ✨ Funcionalidades

- Cadastro de eventos
- Pesquisa por nome e categoria
- Visualização dos detalhes do evento
- Exclusão de eventos
- Sistema de feedback dos participantes
- Página de comentários
- Upload de imagens dos eventos
- Interface responsiva utilizando Bootstrap
- Banco de dados SQLite

---

## 🛠 Tecnologias utilizadas

- Python 3
- Flask
- SQLite
- HTML5
- CSS3
- Bootstrap 5

## 🚀 Como executar

### Instalar dependências

```bash
pip install -r requirements.txt
```

### Executar o projeto

```bash
python app.py
```

A aplicação estará disponível em:

```
http://127.0.0.1:5000
```
